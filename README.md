#### Libraries used for learning threejs

- NodeJS
(Node.js)[https://nodejs.org/]

- ThreeJS and build tool Vite using terminal.
```
# three.js
npm install --save three

# vite
npm install --save-dev vite
```

- Run program using `npx vite` in the terminal.
